What is Hax?

Hax is a computer project developed by Chroma that consists of enclosing a functional program in a .bat script.

Even though, in the future, it is planned to enlarge the capabilities of the program by including more programming languages, right now Hax is the lightest multitool on the market, thanks to Windows batch file scripting. Discover the tools programmed in Hax, and how a simple batch can do so much while still following the instructions of MSDos. Weeks... months... years of programming have made all this possible in one colorful script.
